This project presents the design and development of a supermarket sales management system aimed at improving the efficiency of daily operations and decision-making. The system integrates core functionalities such as sales processing, inventory tracking, customer management, and employee performance monitoring into a single platform. Built using MySQL, Python, and SQL, it ensures reliable data storage, fast query execution, and scalability for future growth. The system supports various analytical queries, including tracking stock levels, identifying expired products, analyzing customer purchases and debts, evaluating employee performance, and generating financial insights such as total revenue and taxes. By combining automation with a user-friendly interface, the project helps reduce manual work, enhances accuracy, and provides meaningful insights that support better management of the supermarket.
